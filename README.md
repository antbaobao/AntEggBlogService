# AntEggBlogService

> Vue.js+Egg.js+Mongodb的前后端分离的个人博客。

> 博客地址：[ANT](http://120.77.219.106/#/)

## 主要技术栈
* egg、mongoose、jwt鉴权

## QuickStart

克隆远程库
```bash
git clone git@github.com:antbaobao/AntEggBlogService.git
```
安装依赖
```bash
cd AntEggBlogService
npm i
```
运行
```bash
npm run dev
```

### Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```
